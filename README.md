Este projeto é um site institucional moderno e responsivo desenvolvido para uma empresa fictícia de tecnologia chamada **Nexcore Solutions**. A página foi criada como parte da disciplina de **Introdução ao Desenvolvimento Front-end** para demonstrar o domínio de componentes do **Bootstrap 5**.

## 📌 Sobre o Projeto

O site segue o modelo de *Single Page Application* (página única) e foi projetado para oferecer uma experiência de usuário fluida em dispositivos móveis e desktops. O tema escolhido foi **Empresa de Tecnologia**, focando em serviços de desenvolvimento de software e infraestrutura.

## 🚀 Requisitos da Atividade Implementados

De acordo com as especificações da atividade, o projeto inclui:

* **Navbar Responsiva:** Menu que se adapta a diferentes tamanhos de tela, contendo links de navegação e botão de contato.
* **Carousel Dinâmico:** Apresentação visual com 3 imagens, indicadores de navegação e legendas (captions).
* **Accordion (FAQ):** Seção de dúvidas frequentes com sistema de colapso, permitindo apenas um item aberto por vez para melhor organização.
* **Modal de Contato:** Janela flutuante acionada por botão, contendo um formulário completo.
* **Formulário de Validação:** Campos para Nome, E-mail e Mensagem, integrados dentro do Modal.
* **Rodapé Informativo:** Contém os dados do desenvolvedor e informações acadêmicas conforme solicitado.

## 🛠️ Tecnologias Utilizadas

* [Bootstrap 5.3.3](https://getbootstrap.com/) - Framework CSS principal.
* [Bootstrap Icons](https://icons.getbootstrap.com/) - Biblioteca de ícones.
* [Google Fonts](https://fonts.google.com/) - Tipografia personalizada.
* [Unsplash](https://unsplash.com/) - Imagens de alta resolução via CDN.
